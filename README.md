# 🎯 김비서 (KimSecretary)

당신의 업무 효율화를 돕는 **AI 기반 업무 관리 대시보드**

## ✨ 주요 기능

- 📋 **할 일 관리** - 우선순위별 업무 추적
- 📅 **주간 일정** - 한눈에 보는 일정 관리
- 📈 **프로젝트 진행률** - 실시간 진행 현황
- 💰 **매출 분석** - 매출 추이 및 통계
- 📊 **회의록 정리** - 회의 결과 및 액션 아이템 관리
- 🔄 **업무 프로세스** - 5단계 프로세스 시각화

## 🎨 디자인 특징

- **글래스모피즘** - 현대적이고 우아한 UI
- **다크/라이트 모드** - 사용자 선호도에 따른 테마
- **반응형 디자인** - 모바일, 태블릿, 데스크톱 지원
- **부드러운 애니메이션** - 직관적인 상호작용

## 📂 파일 구조

```
.
├── index.html              # 메인 소개 페이지
├── dashboard.html          # 업무 대시보드 (메인)
├── meeting-result.html     # 회의 결과 리포트
├── report.html             # 웹사이트 분석 리포트
├── chart.html              # 매출 현황 차트
├── diagram.svg             # 업무 프로세스 다이어그램
├── .gitignore              # Git 제외 파일
└── README.md               # 이 파일
```

## 🚀 배포

### GitHub
```bash
# 저장소 클론
git clone https://github.com/YOUR_USERNAME/kim-secretary.git

# 로컬에서 실행
# 간단히 index.html을 브라우저에서 열기
```

### Vercel 배포
1. [Vercel](https://vercel.com)에 가입
2. GitHub 저장소 연결
3. 자동으로 배포됨
4. 모든 HTML 파일이 라우팅됨

## 🔧 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: CSS Variables, Glassmorphism
- **Storage**: localStorage (테마 설정)
- **Deployment**: Vercel, GitHub Pages

## 📊 데이터 소스

- 업무 목록: CSV 기반
- 주간 일정: 텍스트 기반
- 프로젝트 현황: CSV 분석
- 매출 데이터: 통계 기반

## 🌙 기능 상세

### 다크/라이트 모드
- 우측 상단 토글 버튼으로 전환
- localStorage에 자동 저장
- 모든 페이지에서 유지

### 상단 네비게이션
- 📊 대시보드 (현재 페이지)
- 📋 회의록
- 📈 매출 현황
- 🔄 업무 프로세스
- 📊 사이트 분석

### 대시보드 섹션
1. **할 일 목록** - 체크박스, 우선순위 색상 표시
2. **주간 일정** - 월~금 상세 일정
3. **프로젝트 진행률** - 6개 프로젝트 추적
4. **매출 요약** - 4개 핵심 지표

## 💡 사용 예시

### 업무 추적
1. index.html에서 대시보드로 이동
2. 할 일 항목 체크박스로 완료 표시
3. 우선순위별로 업무 구분

### 회의 관리
1. 네비게이션에서 "회의록" 클릭
2. 회의 정보, 논의 내용, 액션 아이템 확인
3. 인쇄로 기록 보관

### 진행 현황 모니터링
1. 대시보드의 프로젝트 진행률 확인
2. 실시간 매출 현황 추적
3. 업무 프로세스로 전체 흐름 이해
